## 🚀 How to Run

1. تحميل المشروع:
git clone https://github.com/SaharIbrahem114/my-project.git

2. الدخول للمجلد:
cd my-project

3. تثبيت الحزم:
composer install

4. نسخ ملف البيئة:
cp .env.example .env

5. توليد المفتاح:
php artisan key:generate

6. تشغيل المشروع:
php artisan serve
